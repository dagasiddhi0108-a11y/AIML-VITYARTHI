LANGUAGE TRANSLATOR (PYTHON)

OVERVIEW

This project is a Python-based text translation tool that allows users to translate text into multiple languages using the googletrans library. It provides features such as basic translation, batch translation, file translation, language detection, translation history, and a user-friendly menu system. The goal of the project is to make language translation simple, fast, and accessible without requiring any paid API keys.

FEATURES

This includes multiple translation modules like-

Multi Languuage Translation

Batch Translation-Can translate multiple sentences at once

User-Friendly menu system-Easy to understand interface

No API Key Required- It works without needing any paid API key or registration.

PROJECT STRUCTUR

Since the project is smaall,all logic is inside a single python file.

No extra folders or modules are required

TECHNOLOGES USED

Programming Language

Python 3.x

Libraries / Packages

googletrans==4.0.0-rc1

Steps to Install & Run the Project

Install Python
Download and install Python 3.x from the official website.

Install Required Library
Open terminal / CMD and run:

pip install googletrans==4.0.0-rc1

Save the Project Codi
Save your Python program as:

translator.py

Run the Program
Open terminal in the project folder and run: python translator.py
